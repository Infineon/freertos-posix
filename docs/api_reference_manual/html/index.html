FreeRTOS-POSIX
FreeRTOS POSIX library for Infineon MCUs

Introduction

This library contains a posix API implementation on top of FreeRTOS. It implements a small subset of posix specification.

FreeRTOS POSIX API Reference

Following APIs are supported in freertos-posix library. Please refer the POSIX page for API details : API Reference
